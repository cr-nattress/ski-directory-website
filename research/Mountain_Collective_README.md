# Mountain Collective Pass — Resort List (2025–26-ish snapshot)

**Official info & purchase:** https://mountaincollective.com/

> ⚠️ Resort lineups can change each season. Always confirm current details on the official site before buying or planning a trip.

## Overview

The **Mountain Collective** is a global multi-resort pass that gives you:
- 2 days at each Collective resort
- 50% off additional days at those resorts
- No blackout dates (per current marketing; check fine print each year)

This README focuses on the resorts that are part of the Collective and are relevant to North American skiers, but it also includes the international partners for completeness.
