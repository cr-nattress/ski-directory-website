# L’EST GO Card — Resort List

**Official info:** https://www.easterntownships.org/lestgo  
(Purchase/management portal: https://lestgo.ca/)

The **L’EST GO card** is a multi-ticket product for Quebec’s Eastern Townships. You buy a card loaded with 2–8 days and can redeem those days at any of the participating mountains.

---

## Included Resorts (Eastern Townships, Québec, Canada)

- Mont-Orford — Orford, Québec
- Mont SUTTON — Sutton, Québec
- Owl’s Head — Mansonville / Potton, Québec
- Bromont, montagne d’expériences — Bromont, Québec

---

## How It Typically Works

- Load the card with a fixed number of **day tickets** (2–8).
- Use those days at **any combination** of the four participating resorts.
- Days are usually valid **anytime during the season**, with limits on carry-over and some admin fees if you roll days forward.

Exact pricing, quantities, and conditions change each season, so always check the official Eastern Townships L’EST GO page before relying on this data.
