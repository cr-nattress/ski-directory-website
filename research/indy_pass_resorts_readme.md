# Indy Pass Resort List

This README contains a consolidated, structured list of ski resorts included on the **Indy Pass**.  
The Indy Pass generally includes **Independently-owned ski areas** across the U.S., Canada, Japan, and Europe.  
The list below includes the core Indy Pass network as of the most commonly published public listings.

> Note: Indy Pass includes 180+ resorts, but the most publicly stable list is closer to ~120 primary destinations.  
> This README includes those consistently listed across verified public sources.
