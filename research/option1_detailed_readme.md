
# Option 1 — Ski Oracle Landing Page Concept
## Hybrid Detailed README (Design + Technical)

### 1. Overview
The Ski Oracle concept represents a bold, AI‑forward landing page designed mobile‑first. It uses a striking hero question, dynamic hooks, and embedded intelligence to pull users immediately into an interactive ski experience.

### 2. Core Goals
- Deliver instant user engagement through a question‑driven hero.
- Provide a frictionless mobile-first scroll experience.
- Integrate AI at every layer of interaction.
- Present live data and predictive insights.
- Offer modular sections that adapt to user intent.

### 3. Hero Concept
- Large central question: “Where should you ski this week?”
- Rotating AI-powered prompts.
- One-tap access to a full-screen Ski Oracle assistant.

### 4. Mobile Layout Structure
1. Hero + AI prompt
2. Live statewide ski tiles (snowfall, lifts, storms)
3. Predictive insights (best snow, lift delays, terrain openings)
4. STR lodging slider with walk time, ski-in/out tags
5. AI resort finder
6. Snow dashboard

### 5. Technical Design Considerations
- Must scale beautifully to tablet and desktop.
- Dynamic modules call backend APIs for snow, lifts, conditions.
- AI hooks fed from aggregate root data model.
- Fully SEO + LLM compatible.

### 6. Visual Identity
- Dark alpine gradient
- Large bold typography
- Subtle snow particle motion
- Light interactive glow on elements
