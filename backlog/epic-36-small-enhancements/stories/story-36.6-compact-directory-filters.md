# Story 36.6: Compact Filter Display on Resort Directory

## Description

Find a more compact way to display filters on the resort directory page to reduce visual clutter and improve usability, especially on mobile.

## Acceptance Criteria

- [ ] Filters take up less vertical space
- [ ] Filters remain easily accessible and usable
- [ ] Mobile and desktop layouts optimized
- [ ] Filter state clearly visible when active

## Design Options to Explore

1. **Horizontal pill/chip layout** - Filters as inline chips
2. **Dropdown filter button** - Single button that expands to show filters
3. **Collapsible filter section** - Expandable panel
4. **Sticky filter bar** - Compact fixed bar at top

## Technical Notes

- File: `apps/v1/app/directory/page.tsx` and filter components
- Consider mobile-first approach
- Maintain accessibility

## Effort

Medium (2-4 hours)
