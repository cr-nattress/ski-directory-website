# Story 36.14: Dashboard Cards Filter/Sorting Investigation

## Description

Investigate and implement improved filtering or sorting options for dashboard resort cards. Consider alphabetical sorting (A-Z / Z-A), category filters, or alternative organization methods.

## Research Questions

- [ ] What sorting/filtering options would be most useful?
- [ ] How do users currently browse dashboard cards?
- [ ] What are competitors doing?
- [ ] Should filters persist across sessions?

## Options to Consider

### Sorting
1. **Alphabetical (A-Z / Z-A)** - Simple, predictable
2. **By state/province** - Geographic grouping
3. **By distance** (if location known) - Personalized
4. **By recent snow** - Timely, relevant
5. **Randomized** - Discovery-focused

### Filtering
1. **Pass affiliation** - Epic, Ikon, Indy, Local
2. **State/province** - Geographic
3. **Features** - Night skiing, terrain park, etc.
4. **Skill level** - Beginner-friendly, expert terrain

### Alternative Approaches
1. **Search-first** - Prominent search bar
2. **Map-first** - Geographic exploration
3. **Quiz-based** - "Find your perfect resort"

## Acceptance Criteria

- [ ] Research completed and documented
- [ ] Recommendation made with rationale
- [ ] If implementing: chosen solution works on mobile and desktop

## Effort

Medium (Research: 2 hours, Implementation: 2-4 hours)
