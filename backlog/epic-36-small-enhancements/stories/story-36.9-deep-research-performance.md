Conduct comprehensive research on performance optimization opportunities for the ski resort directory application.

- [ ] Measure current LCP, FID, CLS scores

- [ ] Document optimization opportunities

### 3. Asset Optimization
- [ ] Image loading strategy (GCS images)
- [ ] Font loading optimization
- [ ] JavaScript bundle analysis
- [ ] CSS optimization

### 4. Caching Strategy
- [ ] API response caching
- [ ] CDN configuration (Netlify)
- [ ] Browser caching headers
- [ ] Service worker opportunities

### 5. Database/API
- [ ] Supabase query optimization
- [ ] Connection pooling
- [ ] Data denormalization opportunities

## Deliverables

- [ ] Performance audit document

- [ ] Estimated impact for each recommendation
- [ ] Implementation effort estimates

- Lighthouse / PageSpeed Insights

- Next.js Analytics
- Bundle Analyzer
- WebPageTest
