# Story 36.11: Compact Intelligent Categories on Mobile Dashboard

## Description

Create a more compact version of the intelligent categories display on the dashboard for mobile devices to reduce scrolling and improve discoverability.

## Acceptance Criteria

- [ ] Categories display more compactly on mobile
- [ ] Categories remain easily tappable (min 44px touch targets)
- [ ] Smooth scroll behavior for horizontal scrolling (if applicable)
- [ ] Visual hierarchy maintained
- [ ] Active/selected state clearly visible

## Design Options

1. **Horizontal scroll chips** - Single row with horizontal scroll
2. **2-column grid** - More categories visible per screen
3. **Collapsible categories** - Show top 4, expand for more
4. **Icon-only mode** - Icons with text on tap/hover

## Technical Notes

- Consider using CSS scroll-snap for horizontal scrolling
- Maintain accessibility for screen readers
- Test on various mobile screen sizes

## Effort

Medium (2-4 hours)
