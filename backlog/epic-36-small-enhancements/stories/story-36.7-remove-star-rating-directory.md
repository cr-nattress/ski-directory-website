# Story 36.7: Remove Star Rating from Resort Directory

## Description

Remove the star rating display from resort cards in the resort directory to simplify the UI and reduce visual noise.

## Acceptance Criteria

- [ ] Star ratings no longer displayed on resort cards in directory
- [ ] Card layout adjusted to fill space appropriately
- [ ] No visual artifacts from removal

## Rationale

- Star ratings may not provide significant value in directory context
- Simplifies card design
- Reduces cognitive load when browsing

## Technical Notes

- Update ResortCard component or directory-specific card variant
- Consider keeping rating data in case of future use

## Effort

Small (30 minutes)
