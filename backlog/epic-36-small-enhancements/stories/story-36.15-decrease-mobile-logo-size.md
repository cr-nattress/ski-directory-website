# Story 36.15: Decrease Mobile Logo Size by 10%

## Description

Reduce the logo size on mobile devices by approximately 10% to free up screen real estate and improve the mobile header layout.

## Acceptance Criteria

- [ ] Logo size reduced by ~10% on mobile breakpoints
- [ ] Logo remains crisp and readable
- [ ] Header layout remains balanced
- [ ] Desktop logo size unchanged (if appropriate)

## Technical Notes

- Update logo component or CSS
- May need to adjust header padding/margins
- Test on various mobile screen sizes

## Effort

Small (30 minutes)
