# Story 36.1: Collapsible Ski Links Sections

## Description

Make the sections on the Ski Links page collapsible/expandable to improve navigation and reduce scrolling, especially on mobile devices.

## Acceptance Criteria

- [ ] Each category section on the Ski Links page can be collapsed/expanded
- [ ] Sections remember their state during the session
- [ ] Visual indicator shows expanded/collapsed state (chevron icon)
- [ ] Smooth animation for expand/collapse transition
- [ ] All sections expanded by default on first load

## Technical Notes

- File: `apps/v1/app/ski-links/page.tsx`
- Use existing accordion pattern or implement simple toggle state
- Consider using `details/summary` HTML elements for accessibility

## Effort

Small (1-2 hours)
